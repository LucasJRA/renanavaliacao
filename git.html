O que são?

Git é um sistema de controle de versão para rastrear alterações em projetos. GitHub é uma plataforma que utiliza o Git para hospedar, colaborar e gerenciar projetos de software de forma eficiente.

Passo a passo para configurar o Git local:

  • 1- Confira se o git está instalado utilizando o comando "git --versions"
  • 2- Caso esteja instalado abra sua pasta de documento utilizando o comando "ls"
  • 3- Após isso você precisa configurar seu git com os seguintes comandos: 'git config --global user.name "seunome"' logo após este comando utilize o 'git config --global user.email "seuemail"'
  • 4- Após isso você deve criar uma pasta para armazenar seus documentos, para isso utilize o seguinte comando 'mkdir "nome da pasta"'
  • 5- Depois de criar sua pasta você precisa verificar se ela foi criada com sucesse, utilize o comando 'ls' para isso
  • 6- Após ver se sua pasta está criada você pode acessar ela utilizando o comando 'cd "nome da pasta"'
  • 7- Após isso você pode criar um repositório para o seu git, utilize o seguinte comando 'git init'
  • 8- Por fim você pode abrir a sua pasta no VSCode (Visual Studio Code) utilizando o comando 'code .'

Passo a passo para configurar sua chave SSH:

  • 1- Primeiro verifique se existe uma chave utilizando o comando 'ssh.ls -al ~/.ssh'
  • 2- Após verificar crie uma nova chave (ID), para isso utilize o comando 'ssh-keygen -t ed25519 -C "your_email@example.com"'
  • 3- Depois inicialize o agente-ssh com o seguinte comando 'eval "$(ssh-agent -s)"'
  • 4- Agora adicione sua chave ssh ao agente, para isso utilize este comando ssh-add ~/.ssh/id_ed25519 e copie sua chave SSH, para isso pode utilizar este comando: 'clip < ~/.ssh/id_ed25519.pub'
  • 5- Após isso vá até seu github, procure por settings, depois SSH and GPG keys, após isso procure por New SSH key, coloque um titulo para seu branch e aperte ctrl+v ou clique com o botão direito e clique em 'colar' para adicionar sua chave SSH, após isso apenas confirme
  • 6- Por fim para saber se está tudo certo você pode testar a conexão utilizando o comando 'ssh -T git@github.com', após utilizar este comando vai ser perguntado a você quer colocar sua chave no GitHub, caso queira digite 'yes' caso contrário digite 'no'

Passo a passo para criar e clonar um repositório

  • 1- Primeiro você deve acessar sua conta no GitHub e procure pela opção 'new' e clique nela
  • 2- Após isso você insere as informações solicitadas e altera as desejadas e clique no botão 'create repository'
  • 3- Depois vá para a pagina inicial de seu repositório e procure pelo botão '<> code' clicando nele vai algumas opções como 'HTTPS' 'SSH' e 'GitHub CLI', clique em 'SSH' e após isso copie o que é mostrado no campo abaixo, deve ser algo parecido com "git@github.com:seunomedeusuario/nomedorepositorio.git"
  • 4- Após isso e com o git já configurado em sua máquina utilize o comando 'git clone' com o caminho que você copiu no final dele, deve ficar da seguinte forma 'git clone git@github.com:seunomedeusuario/nomedorepositorio.git', após utilizar o comando basta esperar ele ser clonado para sua máquina e depois você pode acessa-lo sem problemas
