Como utilizar o GitHub Pages?

Para utilizar o github Pages você precisa apenas ter um repositório com algum conteudo, você pode hospedar diversas paginas estáticas utilizando ele e em alguns casos você pode hospedar páginas em react também, porém não é possível hospedar um site dinâmico muito grande por esta função.

Adicionar um repositório ao GitHub Pages:


  • 1- Primeiro você precisa estar na página do repositório que deseja adicionar

  • 2- Após isso você procura pela opção settings ou opções (DENTRO DO SEU REPOSITÓRIO)

  • 3- Após acessar esta opção procura por "pages" e clique nela

  • 4- Após isso selecione a branch que deseja subir e se tiver algum dominio personalizado você pode adiciona-lo ao seu site.
